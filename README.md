# Dev

1. Clone project
2. Copy ```.env.template``` and rename to ```.env```
3. Run
```
yarn install
```
4. Up image (Docker desktop)
```
docker-compose up -d
```
5. Up the Nest backend
```
yarn start:dev
```
6. Visit site
```
localhost:3000/graphql
```
